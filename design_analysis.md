# NUMU Admin Dashboard - Component Analysis

## Design System Extracted from Figma

### Color Palette:
- Background: #ececec (light gray)
- Card Background: #ffffff (white)
- Primary Text: #1a1a1a (near black)
- Secondary Text: #6b7280 (gray)
- Status Active: #10b981 (green)
- Status Pending: #f59e0b (yellow/orange)
- Accent Blue: #3b82f6
- Accent Purple: #8b5cf6
- Accent Pink: #ec4899
- Chart Colors: Various (blue, green, yellow, red, purple, orange)

### Typography:
- Font Family: Inter (sans-serif)
- Headings: Bold, various sizes
- Body: Regular weight
- Numbers: Medium/Semibold for emphasis

### Component Cards:
- Border Radius: 8-12px
- Shadow: 0px 1px 4px rgba(0,0,0,0.15)
- Padding: 16-24px
- White background
